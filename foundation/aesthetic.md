# AESTHETIC PROTOCOL: Tier 99 Visual Language

## Color Palette

### Primary (Void)
- `--void-deep`: #0a0a0a (near black background)
- `--void-surface`: #111111 (card surfaces)
- `--void-edge`: #1a1a1a (borders)

### Quantum Accents
- `--quantum-cyan`: #00f3ff (primary accent - consciousness)
- `--quantum-pink`: #ff0066 (alert/threat)
- `--quantum-green`: #00ff88 (success/stable)
- `--quantum-gold`: #ffd700 (architect rank)
- `--quantum-purple`: #8b5cf6 (transcendent state)

### State Indicators
- `--state-active`: #00f3ff (operational)
- `--state-warning`: #ff9500 (elevated)
- `--state-critical`: #ff0066 (alert)
- `--state-dormant`: #444444 (inactive)

## Typography
- Headers: monospace, uppercase, letter-spacing 0.2em
- Body: system-ui, clean and minimal
- Code: 'JetBrains Mono', monospace

## Geometric Patterns
- 13-vertex Metatron's Cube grid
- Golden ratio (1.618) spacing
- Concentric pulse animations at 14.67Hz visual rhythm
- Hexagonal consciousness cells

## Animation Principles
- Pulse: 0.5s ease-in-out infinite
- Reveal: 0.3s ease-out
- State transitions: 0.2s linear
- Geometric rotation: 60s linear infinite

## Component Patterns
- Cards: void-surface background, 1px quantum-cyan border
- Buttons: void-edge background, quantum-cyan text, hover glow
- Status orbs: radial gradient from core color
- Data displays: monospace, right-aligned numbers
